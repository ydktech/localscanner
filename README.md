# WhereToGo

위치 기반 주변 정보 제공 모바일 앱

## 기술 스택
- Flutter
- Dart
- Provider / Riverpod
- Firebase / Supabase
- Google Maps API

## 핵심 기능
- 실시간 위치 기반 서비스
- 주변 맛집/카페/관광지 정보
- 거리별 필터링 및 정렬
- 장소 상세 정보 및 리뷰
- 즐겨찾기 및 방문 기록

## 실행
```bash
flutter create wheretogo
cd wheretogo
flutter run
```

## 테스트
```bash
flutter test
```
