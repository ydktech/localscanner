# WhereToGo

위치 기반 주변 정보 제공 모바일 앱

## 기술 스택
- Flutter
- Dart
- Provider / Riverpod
- Firebase / Supabase
- Google Maps API

## 핵심 기능
- 실시간 위치 기반 서비스
- 주변 맛집/카페/관광지 정보
- 거리별 필터링 및 정렬
- 장소 상세 정보 및 리뷰

## 웹으로 디버깅
- playwright-mcp

## 기술 지침
- 반드시 기능 구현이 완료됐는지 flutter web으로 빌드해서 playwright-mcp로 UI의 구성이나 버그 체크를 하세요.
